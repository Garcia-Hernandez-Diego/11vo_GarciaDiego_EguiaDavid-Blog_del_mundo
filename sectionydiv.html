Aprendiendo lo básico de HTML

¿Para qué sirve la etiqueta section?

La etiqueta section se utiliza en html para separar la información en secciones. Este lo hace de forma mayor, dentro de estos pueden haber etiquetas div.

¿Para qué sirve la etiqueta div?

La etiqueta div se utiliza en html para una separación de información dentro de la etiqueta section. Este no tiene ningún cambio en el archivo.

¿Para qué sirve la etiqueta main?

La etiqueta main sirve para representar todo el contenido principal de nuestro blog. Se pone en el body del documento.

¿Para qué sirve la etiqueta aside?

La etiqueta aside representa el contenido de la página separado del contenido principal en un costado de la página.

¿Para qué sirve la etiqueta article?

La etiqueta article sirve para especificar contenido que puede ser distribuido independientemente del resto del sitio o aplicación.
