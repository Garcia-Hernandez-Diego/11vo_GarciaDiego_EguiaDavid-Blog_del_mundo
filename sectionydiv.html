¿Para qué sirve la etiqueta section?

La etiqueta section se utiliza en html para separar la información en secciones. Este lo hace de forma mayor, dentro de estos pueden haber etiquetas div.

¿Para qué sirve la etiqueta div?

La etiqueta div se utiliza en html para una separación de información dentro de la etiqueta section. Este no tiene ningún cambio en el archivo.
